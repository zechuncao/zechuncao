Overview

We implemented the Triangle class with three sides in Lab 1. In a triangle, the sum of any two sides is greater than the other side. The Triangle class must adhere to this rule. Download Lab7.java from Blackboard, create the IllegalTriangleException class, and modify the constructor of the Triangle class to throw an IllegalTriangleException object if a triangle is created with sides that violate the rule, as follows: 

    /** Construct a triangle with the specified sides */
    public Triangle(double side1, double side2, double side3)
      throws IllegalTriangleException {
      // Implement it
    }
