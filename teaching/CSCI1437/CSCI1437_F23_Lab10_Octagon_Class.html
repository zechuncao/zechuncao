Overview

Write a class named Octagon that extends GeometricObject and implements the Comparable and Cloneable interfaces. Assume that all eight sides of the octagon are of equal length. The area can be computed using the following formula:

\begin{equation} % arbitrary environments, area = (2+4/\sqrt{2})*side*side % even tables, figures \end{equation}

Tasks

  1. Write a test program that creates an Octagon object with side value 5 and displays its area and perimeter.
  2. Create a new object using the clone method and compare the two objects using the compareTo method.
  3. Use java.util.Arrays.sort to sort objects of Octagon based on area ascendingly.
