Task 1. Data Fields

  • Define a String data field names name to denote the name of the triangle, with no default value assigned.
  • Define three double type data fields named side1, side2, and side3 with default values of \(1.0\) to denote three sides of triangle.

Task 2. Constructors

  • Constructor #1: A no-arg constructor that creates a triangle with default values for side1, side2, and side3.
  • Constructor #2: A constructor to create a triangle with the specified values for name, side1, side2, and side3.

Task 3. Methods

  • Three public accessor methods (getters) for three data fields side1, side2, and side3. For example: 

    public double getSide1() {
     return side1;
}
  • A method named getPerimeter() that computes and returns the perimeter of this triangle.
  • A method named getArea() that computes and returns the area of this triangle.

  • A method named toString() that returns a description, String type, for the state of the triangle. Example description shown below: 

    Triangle name: my_triangle
side1 = 1.0
side2 = 2.0
side3 = 2.0
Area = 0.9682
Perimeter = 5

Task 4. TestTriangle Class

Write a test function (main function) in the TestTriangle class, so the test function can test all functionalities in Triangle class. Example test function below: 

public static void main(String[] args) {
        // Create a triangle with no-arg constructor
        Triangle triangle1 = new Triangle();
        System.out.println(triangle1.toString());
        triangle1.setSide1(4.0)
        triangle1.setSide2(4.0)
        triangle1.setSide3(4.0)
        System.out.println(triangle1.toString());


        // Create a ctriangle with the second constructor
        Triangle my_triangle = new Triangle("myTriangle", 1.0, 2.0, 2.0);
        System.out.println(my_triangle.name + " Area: " + my_triangle.getArea());
        System.out.println(my_triangle.name + " Perimeter: " + my_triangle.getPerimeter());
        System.out.println(my_triangle.toString())
    }
