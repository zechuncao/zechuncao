Design a class named Fan with a TestFan class to define a fan and its behavior, similar to the TV example in the lecture. Build the Fan class with requirements below, then write a main function in your TestFan class to test the functionalities.

Task 1. Data Fields

  • Three static int constants names SLOW, MEDIUM, and FAST with the values 1, 2, and 3 to denote the fan speed.
  • A static data field named numOfFans that keeps track of the number of fan objects created.
  • A private int data field named speed that specifies the speed of the fan, the default is SLOW.
  • A private boolean data field named on that specifies whether the fan is on (the default is false).
  • A private double data field named size that specifies the size radius of the fan as its radius (the default is 5, valid range is [1.0, 10.0]).
  • A private String data field named color that specifies the color of the fan (the default is blue).

Task 2. Constructors

  • Constructor #1: A no-arg constructor that creates a fan with default values for all data fields.
  • Constructor #2: A constructor to create a fan with the specified values for speed, on, size, and color.

∗ Note: you need to properly update, i.e., increment by 1, the value stored in numOfFans in both constructors.

Task 3. Methods

  • The public accessor (getters) and mutator (setters) methods for all four data fields.
  • A public method named toString() that returns a string description for the fan. If the fan is on, the method returns the fan speed, color, and size in one combined string. If the fan is not on, the method returns the fan color and radius along with the string “fan is off” in one combined string.

∗ Note: 1. the setter method for the fan’s size needs to check if the argument value is within the valid range; 2. fan’s speed can only be changed when the fan is on; 3. fan’s size and color can only be changed when the fan is off.

Task 4. TestFan Class

Write a test function (main function) in the TestFan class, so the test function can test all functionalities in the Fan class. Write a test program that creates two Fan objects. Assign maximum speed, size=10, color yellow, and turn it on to the first object. Assign medium speed, size=5, color blue, and turn it off to the second object. Display the objects by invoking their toString() method.
